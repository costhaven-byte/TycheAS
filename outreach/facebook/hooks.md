# Facebook Group Value Hooks

Ten ready-to-post hooks, one per vertical, each with a **US** and **UK** variant.
Same core angle as the email A/B test (*missed leads → free audit*), rewritten to
survive group rules: no links in the body, a soft "DM me / drop a 🙋 below" CTA,
and owner-lingo, not agency-speak.

**Rules that keep you un-banned:**
- Post to the group's promo/value thread, or as a normal helpful post — never a raw pitch.
- No external links in the body. Put "DM me" or "comment and I'll send it."
- Move to DM only after they reply or react. Never cold-DM the whole group.
- 5 genuinely-helpful comments for every 1 hook you post.
- Log each post in `data/activity-log.csv` with its `hookId` so `npm run fb:stats` can score it.

Rotate. Don't post the same hook to the same group twice in a month.
